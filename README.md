# org-pivotal

## Summary

org-pivotal is an emacs minor mode to extend org-mode with Pivotal Tracker abilities

## Installing

You will need Emacs 24+, `make` and [Cask](https://github.com/cask/cask) to
build the project.

    cd org-pivotal
    make && make install
